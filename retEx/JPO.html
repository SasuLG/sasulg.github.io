Jeu Java

Ce projet à été réalisé par groupe de 4. Le but est de crée un jeu simple en java, permettant d'apprendre les bases du langage.
Il est destiné aux personnes souhaitant rejoindre l'iut informatique lors des journée portes ouvertes.
Le jeu consiste à déplacer la grenouille dans un labyrinthe (6 différentes avec plusieurs niveau de diffcultés) en modifiant le code afin de la faire atteindre le nénnuphar, le tout en évitant les murs.

Process

1-Trouver une idée de jeu
2- Choix des designs (perso, labyrinthe)
3- Construction des labyrinthes
4- Programmation de toutes les méthodes

Compétences acquises

1- Construction d'un jeu java de a à z
2- Travail en groupe
3- Gestion d'un projet sur plusieurs semaines
