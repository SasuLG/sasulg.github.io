Création de base de donnée

Ce projet consiste à la création d'une base de données à l'aide de WinDesign, pour la gestion de concert et de festival.
Il fallait donc d'abord créer le mcd puis le mlr pour en générer le script. Enfin il ne restait qu'à faire une script de remplissage pour cette base de donnée.
Ce projet se déroulait en binôme.

Process

1- Création MCD et MLR
2- Génération du script et création BD
3- Remplissage de la base de données

Compétences acquises

1- Amélioration programmation Sql
2- Gestion d'un projet sur plusieurs semaines
3- Travail et organisation en groupe
4- Utilisation de WinDesign pour MCD et MLR
